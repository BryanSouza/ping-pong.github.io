# Ping-Pong :ping_pong:
Ping-Pong is a game in which a player hits a ball across the table to score.

## Keyboard Controls
- Player One
  - Move Left: **Arrow Left**.
  - Move Right: **Arrow Right**.
  - Serve the ball: **Arrow Up**.

- Player Two
  - Move Left: **Key A**.
  - Move Right: **Key D**.
  - Serve the ball: **Key S**.
